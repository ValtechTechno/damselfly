* La configuration doit être éditable (une clé peut être présent sur plusieurs environnements)
* La configuration est définie par des couples <clé, valeur>, par environnement et par application (et le cas échéant par version)
* Possibilité de chiffrer des données
* Possibilité de recharger à chaud certaines propriétes de la configuration
